# Анализатор зависимостей пакетов

Простое консольное приложение для анализа графов зависимостей npm пакетов.

## Что умеет программа (Этап 3)

- Строит полный граф зависимостей с учетом транзитивности
- Использует BFS алгоритм без рекурсии для обхода
- Учитывает максимальную глубину анализа
- Обнаруживает циклические зависимости
- Поддерживает тестовый режим с пакетами A, B, C...
- Показывает зависимости в виде дерева или списка
