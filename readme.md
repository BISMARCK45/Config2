# Анализатор зависимостей пакетов

Простое консольное приложение для анализа графов зависимостей npm пакетов.

### Этап 4: Дополнительные операции
- Находит обратные зависимости - какие пакеты зависят от заданного
- Анализирует граф в обоих направлениях
- Демонстрирует работу на тестовых репозиториях
